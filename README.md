# React-Person-Age-Calculator
A simple and beginner-friendly React application that displays a list of persons along with their date of birth and calculated age. This project is built using React and Vite and demonstrates core React concepts like components, props, data mapping, and utility functions.

## Features
- Display person details in a table format
- Calculate age automatically from Date of Birth
- Clean and simple UI
- Uses modern React with Vite
- Easy to understand for beginners

## Technologies Used
- React.js
- Vite
- JavaScript (ES6)
- HTML
- CSS

## Explanation
- `personsData` contains static person information.
- `calculateAge()` function calculates age using JavaScript Date.
- `PersonDetails` component maps data and displays it in a table.
- `App.jsx` renders the main component.
- `main.jsx` is the entry point of the application.

The application shows:
- Name  
- Date of Birth  
- Calculated Age  
Displayed neatly in a table.

## Learning Outcomes
- Understanding React component structure
- Using map() to render lists
- Writing utility functions
- Handling dates in JavaScript
- Project setup using Vite

## Conclusion
This project demonstrates the basic use of React components, data handling, and JavaScript date manipulation to calculate and display age dynamically. It helps in understanding component-based architecture, list rendering using map(), and clean project structuring using Vite. Overall, it is a simple yet effective beginner-level React project for learning core concepts.
